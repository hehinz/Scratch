Data structures and invariants ( via )

This article helped me understand the usefulness of reducing the number of codepaths via the use of sentinel values. Ryan Fleury in [1] introduced me to this concept.

Here is the basic to nil-node that can be used as the sentinel.


typedef struct Node Node;

struct Node {
    int data;
    Node *next;
};
struct Node nil_node {
    .data=0,
    .next=&nil_node;
};

Linked lists are seen as an outdated data structure that happily thrashes the cache, but it has its uses. Not everything need to be put in dictionaries. Linear search is often faster [2].
